# UsendMii Client

[![Build Status](https://travis-ci.org/Crayon2000/UsendMii-Client.svg?branch=master)](https://travis-ci.org/Crayon2000/UsendMii-Client)

This is a UsendMii client application for Wii U.

## Build
Prerequisites:
* devkitpro with devkitPPC
* libogc
* portlibs 

To compile:
```
> make
```
